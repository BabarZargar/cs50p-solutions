# CS50P – Introduction to Programming with Python

This repository contains my solutions and practice work for **CS50P: Introduction to Programming with Python**, a course offered by **Harvard University**.

The course focuses on building strong programming fundamentals using Python, covering topics from basic syntax to file handling, regular expressions, and object-oriented programming.

---

## Course Structure

The repository is organized week-wise, following the CS50P curriculum:

```
week 0 – Functions, Variables
week 1 – Conditionals
week 2 – Loops
week 3 – Exceptions
week 4 – Libraries
week 5 – Unit Tests
week 6 – File I/O
week 7 – Regular Expressions
```

Each folder contains Python programs written as part of problem sets and practice exercises.

## What I Learned

Through this course, I gained hands-on experience with:

* Writing clean and readable code

* Working with files (CSV, text, etc.)
